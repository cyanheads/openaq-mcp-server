# 0.1.1 — 2026-06-15

## Added

- **Public hosted endpoint** at `https://openaq.caseyjhand.com/mcp` — added to `server.json` `remotes[]`, README header, and Getting Started section ([#1](https://github.com/cyanheads/openaq-mcp-server/issues/1) addressed alongside).

## Fixed

- **`openaq_list_countries`** — throws `null is not an object (evaluating 'c.parameters.map')` when the OpenAQ `/v3/countries` API returns a country with `parameters: null`. Guard added at the handler mapping site (`(c.parameters ?? []).map(...)`). `OpenAqCountry.parameters` type updated to `OpenAqParameterRef[] | null` to match actual upstream sparsity. Regression test added. ([#1](https://github.com/cyanheads/openaq-mcp-server/issues/1))
