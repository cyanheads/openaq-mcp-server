# 0.1.0 ‚Äî 2026-06-13

Initial release. Wraps the OpenAQ v3 API (`api.openaq.org/v3`) to expose measured air quality ‚Äî physical-sensor observations from government reference monitors and research-grade sensors worldwide. The ground-truth counterpart to modeled air-quality grids; an empty result means no monitoring coverage, not clean air.

## Added

- **`openaq_find_locations`** ‚Äî find monitoring stations near a point, in a bounding box, or by country; returns id, coordinates, distance, provider, the parameters each station measures, and the data span. Capped-list truncation disclosed via enrichment.
- **`openaq_get_readings`** ‚Äî latest value per sensor at a station, joined against the station's sensor‚Üíparameter‚Üíunit map so every value carries its pollutant and unit. Accepts a `locationId` or `coordinates`+`parametersId` (resolves the nearest station).
- **`openaq_get_measurements`** ‚Äî historical series for one pollutant at one station; resolves the sensor internally, supports `raw`/`hourly`/`daily` aggregation, and spills large series to a DataCanvas (preview + `canvasId` + table name).
- **`openaq_list_parameters`** ‚Äî pollutant + unit catalog with local filtering and a `pollutantsOnly` toggle. The unit-disambiguation reference: the same pollutant has multiple ids for different units (CO is id 4 ¬µg/m¬≥, 8 ppm, 102 ppb).
- **`openaq_list_countries`** ‚Äî country coverage catalog (ISO code, data span, parameters measured) for availability checks before a regional sweep.
- **`openaq_dataframe_query`** / **`openaq_dataframe_describe`** ‚Äî read-only SQL over measurement series staged on a DataCanvas (`CANVAS_PROVIDER_TYPE=duckdb`).
- **Resources** ‚Äî `openaq://location/{locationId}` and `openaq://parameters` mirror the tool output for resource-capable clients.
- **`OpenAqService`** ‚Äî single service over the v3 REST API: `X-API-Key` auth, `withRetry`, and OpenAQ-specific error classification (404 ‚Üí NotFound, the Python-repr 422 body ‚Üí ValidationError, the plain-text 500 backstop ‚Üí ServiceUnavailable).

## Notes

- Units are surfaced verbatim and never converted ‚Äî ¬µg/m¬≥ ‚Üî ppm/ppb conversion is gas- and temperature-dependent.
- Requires a free OpenAQ API key in `OPENAQ_API_KEY`, sent as the `X-API-Key` header.
